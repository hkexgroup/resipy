Example: simple 2D ERI time-lapse survey

1. Open th GUI (pyR2):
	- Option 1: use standalone version
	- Option 2: use command line: 	cd /pyr2/src
									python ui.py

2. Select "Inverse" (checked by default)
3. Check "Time-lapse Survey" checkbox
4. Select working directory: /examples/2D-dc-time-lapse
5. Select data type from drop down menu: Syscal (for this example)
6. Import data by selecting data directory: /examples/2D-dc-time-lapse/data
7. Follow the rest of the instructions from pyR2 manual.
