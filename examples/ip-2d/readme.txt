Example: simple flat 2D IP survey

1. Open th GUI (pyR2):
	- Option 1: use standalone version
	- Option 2: use command line: 	cd /pyr2/src
									python ui.py

2. Select "Inverse" (checked by default)
3. Select working directory: /examples/2d-ip
4. Select data type from drop down menu: Syscal (for this example)
5. Import data: rifleday8.csv
6. Follow the rest of the instructions from ResIPy manual.
